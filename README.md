# Coffee House

A static website for Coffee House, a premium coffee shop located in Pune, India. The site showcases the menu, about the business, shop for coffee products, and contact information.

## Features

- Responsive design
- Multiple pages: Home, Menu, About, Shop, Contact
- Image gallery with swiper
- Menu categories with prices
- Product listings in shop
- Contact form and map integration
- Testimonials from customers
- Footer with links and social media

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Swiper.js for image sliders
- Font Awesome for icons
- Google Fonts

## Setup and Usage

1. Clone or download the project files.
2. Open `cofeeshop.html` in a web browser to view the home page.
3. Navigate through the site using the menu links.
